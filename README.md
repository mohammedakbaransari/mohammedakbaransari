### Current Research

**GlassBox — Runtime Decision Governance Framework**

> *"Every AI-initiated operational action should be inspectable, bounded, and enforceable  
> before execution — not just audited afterward."*

A runtime control layer that intercepts, evaluates, and records every AI-generated
decision before it reaches an enterprise execution system.

| What it does | How |
|---|---|
| Decision-semantic governance | Policy-as-code evaluation at the action boundary |
| Anomaly detection | Z-score statistical detection with rolling per-agent baseline |
| Runaway agent protection | Sliding-window velocity circuit breaker |
| Multi-agent lineage tracking | Full agent chain recorded per decision |
| Regulatory audit trail | Immutable structured records per decision |
